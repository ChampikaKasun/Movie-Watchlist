# 🎬 Movie Watchlist

A movie and TV watchlist web app that lets users browse and search films via
the TMDB API and save their favourites to a personal watchlist. Built to
practise working with external APIs alongside React and Firebase.

## Features
- 🔍 Search and browse movies using the TMDB API
- 🖼️ Responsive poster grid with titles and ratings
- 🔐 User authentication with Firebase Auth
- ⭐ Save and remove movies from a personal watchlist (Firestore)
- 🎞️ Movie detail view with overview, rating, and release date

## Tech Stack
React · Vite · Firebase (Auth + Firestore) · TMDB API
